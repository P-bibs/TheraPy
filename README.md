# TheraPy
Soothing error messages to accompany Python exceptions

## Install
```shell
cargo install thera-py
```

## Usage
`thera-py` is a drop-in replacement for python. If you would normally type
```shell
python my-proj.py
```
to run your program instead type
```shell
thera-py my-proj.py
```
